NBA All Time API

A simple API with the best all time rosters from all 30 NBA teams!

made by: Cole Joffray

Endpoints:

Get a random roster

https://nba-alltime.onrender.com/api/:random

By hitting this URL, the API will serve back data (roster) from a random NBA team. The response will be in json object format. This link is great for making trivia games!

URL: https://nba-alltime.onrender.com/api/:random
